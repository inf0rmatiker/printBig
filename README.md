# printBig
- Prints a given string to console in BIG letters!
- Run with the string you want printed to console, made out of characters you can specify.
### Example: 
* python printBig.py "Hello!" % 
### Result: 
```
   %    %  %%%%%%   %       %        %%%%   %%
   %    %  %        %       %       %    %  %%
   %%%%%%  %%%%%%   %       %       %    %  %%
   %    %  %        %       %       %    %
   %    %  %%%%%%   %%%%%%  %%%%%%   %%%%   %%
```
